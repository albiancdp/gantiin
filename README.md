# Gantiin

**Konverter file online, gratis, dan 100% client-side.**

Upload file → pilih format → download hasil. Tidak ada upload ke server, tidak perlu registrasi, dan tidak ada batasan pemakaian. Semua konversi terjadi langsung di browser Anda.

## Fitur

- **PDF → Teks** — Ekstrak teks dari PDF dan simpan sebagai .txt
- **Konversi Gambar** (Coming Soon) — PNG, JPG, WebP, HEIC
- **Merge PDF** (Coming Soon) — Gabungkan beberapa PDF menjadi satu
- **Split PDF** (Coming Soon) — Pisahkan halaman PDF tertentu
- **Dark Mode** — Nyaman digunakan di malam hari

## Tech Stack

| Teknologi | Keterangan |
|-----------|------------|
| Next.js 16 | App Router, static export |
| React 19 | Server & Client components |
| TypeScript | Strict mode |
| Tailwind CSS v4 | Utility-first CSS |
| shadcn/ui (Base UI) | Komponen UI |
| pdfjs-dist 6 | Ekstraksi teks & thumbnail PDF |
| @jsquash/webp | Encode WebP (fallback) |
| heic-to | Konversi HEIC iPhone |
| Umami | Analytics self-hosted |

## Memulai

```bash
# Install dependencies
npm install

# Copy aset pdf.js
npx tsx scripts/copy-pdf-assets.mjs

# Development server
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000).

## Build & Deploy

```bash
# Build static export
npm run build

# Output di folder out/
# Deploy ke Vercel, Cloudflare Pages, atau static hosting lainnya
```

Project ini dikonfigurasi untuk **static export** (`output: 'export'`), sehingga dapat di-deploy ke berbagai static hosting tanpa server Node.js.

## Struktur Project

```
src/
├── app/            # Pages & layouts
├── components/
│   ├── convert/    # UniversalConverter, ConversionOptions, dll
│   ├── layout/     # Header, Footer, ThemeToggle
│   └── upload/     # DropZone, FilePreview
├── hooks/          # useFileUpload, useConversion
├── lib/
│   ├── conversions/ # Registry + engine per format
│   └── errors.ts   # AppError + error codes
public/
├── pdfjs/          # Standard fonts (generated by postinstall)
└── *.{svg,png,ico} # Assets
```
