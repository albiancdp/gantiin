# Roadmap

## Phase 1: Foundation (Minggu 1-2)
**Goal:** Project setup dan basic UI

### Deliverables:
- [ ] Project scaffolding (Next.js + TypeScript + Tailwind)
- [ ] Shadcn/ui setup
- [ ] Landing page design
- [ ] File upload component
- [ ] Dark/Light mode
- [ ] Basic layout (header, footer, main)

### Dependencies: None
### Risk: Low

---

## Phase 2: Core Features (Minggu 3-4)
**Goal:** Implementasi konversi dasar

### Deliverables:
- [ ] PDF → Text conversion
- [ ] Image format conversion (PNG↔JPG↔WebP)
- [ ] Download mechanism
- [ ] Error handling
- [ ] Mobile responsive
- [ ] Performance optimization

### Dependencies: Phase 1
### Risk: Medium (conversion quality)

---

## Phase 3: Extended Features (Minggu 5-6)
**Goal:** Fitur tambahan untuk value

### Deliverables:
- [ ] PDF merge
- [ ] PDF split
- [ ] Image resize
- [ ] Image compress
- [ ] Batch selection (multiple files)
- [ ] Preview sebelum convert

### Dependencies: Phase 2
### Risk: Medium

---

## Phase 4: Polish & Launch (Minggu 7-8)
**Goal:** Testing, optimasi, dan launch

### Deliverables:
- [ ] Performance optimization
- [ ] SEO optimization
- [ ] Analytics integration (Plausible)
- [ ] Donation button (Saweria)
- [ ] Error boundary
- [ ] Testing (unit + E2E)
- [ ] Documentation
- [ ] Deploy ke Vercel

### Dependencies: Phase 3
### Risk: Low

---

## Phase 5: Growth (Bulan 3-4)
**Goal:** User acquisition dan retention

### Deliverables:
- [ ] PWA support
- [ ] History (localStorage)
- [ ] Keyboard shortcuts
- [ ] Share functionality
- [ ] Blog (content marketing)
- [ ] Social media presence

### Dependencies: Phase 4
### Risk: Low

---

## Phase 6: Monetization (Bulan 5-6)
**Goal:** Revenue generation

### Deliverables:
- [ ] Donation tiers
- [ ] "Support Gantiin" page
- [ ] Sponsor logos
- [ ] Monthly report to donors
- [ ] API for developers (beta)

### Dependencies: Phase 5
### Risk: Medium (adoption)
