# Goals & Objectives

## Business Goals

### Q1 (Bulan 1-2): Launch MVP
1. **Landing page & core converter live** — Minimal 3 jenis konversi (PDF→Text, Image format, PDF merge)
2. **1,000 unique visitors** dalam bulan pertama
3. **Zero server cost** — 100% client-side processing

### Q2 (Bulan 3-4): Growth
1. **5,000 monthly active users (MAU)**
2. **First donation received** — Minimal Rp 100.000
3. **Featured in 1 tech blog/media Indonesia**

### Q3 (Bulan 5-6): Sustainability
1. **10,000 MAU**
2. **Rp 5 juta/bulan donation revenue**
3. **User satisfaction score > 4.5/5**

## Product Goals

### MVP (Minggu 1-8)
1. **3 conversion types** — PDF→Text, Image format convert, PDF merge
2. **Mobile responsive** — Bisa digunakan dengan baik di HP
3. **Performance** — File 10MB dikonversi dalam < 5 detik
4. **Accessibility** — WCAG 2.1 AA compliant

### V2 (Bulan 3-6)
1. **5+ conversion types** — Tambah PDF split, image compress, image resize
2. **Batch processing** — Konversi multiple file sekaligus
3. **History** — Simpan riwayat konversi (localStorage)
4. **PWA** — Bisa di-install seperti native app

### V3 (Bulan 7-12)
1. **10+ conversion types** — Termasuk document conversion
2. **API available** — Untuk integrasi third-party
3. **Premium features** — Opsi donasi dengan fitur tambahan
4. **Localization** — Support bahasa Indonesia & English
