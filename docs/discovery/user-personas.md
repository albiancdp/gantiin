# User Personas

## Persona 1: Rina (Mahasiswa)
- **Role:** Mahasiswi semester 6, Fakultas Ekonomi
- **Goals:** 
  - Mengkonversi jurnal PDF ke Word untuk kutipan
  - Mengkompres gambar untuk tugas presentasi
  - Mengubah format gambar untuk upload ke LMS
- **Frustrations:**
  - Limit harian di situs converter gratis
  - Iklan yang menutupi tombol download
  - Khawatir file tugas tersebar di internet
- **Technical Level:** Medium — familiar dengan browser, tapi tidak tech-savvy
- **Key Needs:**
  1. Gratis tanpa registrasi
  2. Cepat dan mudah digunakan
  3. Privasi file terjamin

## Persona 2: Budi (Pekerja Kantoran)
- **Role:** Staff Marketing di perusahaan manufaktur
- **Goals:**
  - Mengkonversi PDF laporan ke format yang bisa diedit
  - Mengubah ukuran gambar untuk email newsletter
  - Merge beberapa PDF menjadi satu dokumen
- **Frustrations:**
  - Harus install software tambahan di laptop kantor
  - Koneksi lambat membuat upload file lama
  - Tidak bisa konversi file besar
- **Technical Level:** Low-Medium — bisa pakai browser, tidak paham teknis
- **Key Needs:**
  1. Tidak perlu install software
  2. Bisa digunakan di browser kantor
  3. Hasil konversi berkualitas

## Persona 3: Dina (Freelancer/Content Creator)
- **Role:** Freelance designer, content creator Instagram
- **Goals:**
  - Mengkonversi format gambar untuk berbagai platform
  - Mengkompres gambar tanpa mengurangi kualitas
  - Mengubah background gambar
- **Frustrations:**
  - Tool berbayar terlalu mahal untuk penghasilan tidak tetap
  - Tool gratis hasilnya jelek (blur, artifact)
  - Tidak ada fitur batch processing
- **Technical Level:** Medium-High — paham format file, tapi butuh yang practical
- **Key Needs:**
  1. Kualitas konversi tinggi
  2. Multiple format support
  3. Cepat untuk banyak file

## Persona 4: Ahmad (Small Business Owner)
- **Role:** Pemilik toko online, jualan di Shopee/Tokopedia
- **Goals:**
  - Mengkonversi foto produk ke format yang dibutuhkan marketplace
  - Mengkompres gambar agar tidak melebihi batas upload
  - Mengubah resolusi gambar untuk berbagai kebutuhan
- **Frustrations:**
  - Harus bolak-balik ke warnet untuk konversi file
  - Tidak paham format file yang berbeda
  - Malas install aplikasi baru di HP
- **Technical Level:** Low — hanya bisa pakai browser dan aplikasi dasar
- **Key Needs:**
  1. Sangat mudah digunakan (tidak perlu paham teknis)
  2. Bisa diakses dari HP
  3. Hasil langsung siap pakai
