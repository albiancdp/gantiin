# User Journey Map

## Scenario 1: First-Time User — Konversi PDF ke Word

### Stages

| Stage | Actions | Thoughts | Emotions | Touchpoints |
|-------|---------|----------|----------|-------------|
| 1. Awareness | Mencari "konversi pdf ke word gratis" di Google | "Butuh tool yang gratis dan gak ribet" | Frustrated (dari pengalaman sebelumnya) | Google Search |
| 2. Discovery | Klik hasil pencarian, melihat landing page Gantiin | "Wah, interface-nya bersih, gak ada iklan" | Hopeful, curious | Landing page |
| 3. Upload | Drag & drop file PDF ke area upload | "Gampang banget, gak perlu registrasi" | Surprised, relieved | File upload area |
| 4. Convert | Klik tombol "Konversi" | "Cepet banget konversinya" | Satisfied | Convert button |
| 5. Download | Klik tombol download hasil Word | "Hasilnya bagus, format gak rusak" | Happy | Download button |
| 6. Return | Bookmark situs untuk konversi berikutnya | "Gantiin jadi go-to tool konversi saya" | Loyal | Bookmark |

## Scenario 2: Repeat User — Kompres Gambar untuk Email

### Stages

| Stage | Actions | Thoughts | Emotions | Touchpoints |
|-------|---------|----------|----------|-------------|
| 1. Need | Perlu kompres gambar untuk attachment email | "Gambarnya kebesaran, gak muat di email" | Mildly frustrated | Email client |
| 2. Access | Buka Gantiin dari bookmark | "Gak perlu cari lagi, langsung buka Gantiin" | Confident | Bookmark |
| 3. Upload | Drag & drop gambar | "Bisa pilih kompresi berapa persen" | In control | File upload |
| 4. Adjust | Atur level kompresi | "Preview-nya bagus, bisa lihat hasil sebelum download" | Satisfied | Settings panel |
| 5. Download | Download gambar yang sudah dikompres | "Ukuran turun 70%, kualitas tetap oke" | Impressed | Download |
| 6. Share | Cerita ke teman tentang Gantiin | "Lu harus coba Gantiin, gratis dan bagus" | Advocate | Word of mouth |

## Scenario 3: Mobile User — Konversi di HP

### Stages

| Stage | Actions | Thoughts | Emotions | Touchpoints |
|-------|---------|----------|----------|-------------|
| 1. Need | Butuh konversi gambar saat di jalan | "Gimana cara konversi gambar di HP?" | Anxious | Mobile browser |

## Pain Points & Opportunities

### Pain Points di Journey Ini:
1. **Discovery:** User sudah frustasi dari pengalaman sebelumnya dengan tool lain
2. **Trust:** User skeptis tool gratis benar-benar gratis
3. **Quality:** User takut hasil konversi jelek
4. **Return:** User perlu alasan untuk kembali menggunakan Gantiin

### Opportunities:
1. **First Impression:** Landing page harus langsung meyakinkan
2. **Zero Friction:** Tidak ada langkah registrasi atau login
3. **Quality Proof:** Tampilkan preview hasil konversi
4. **Retention:** History konversi (localStorage) untuk user yang kembali
