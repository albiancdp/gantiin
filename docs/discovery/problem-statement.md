# Problem Statement

## Current Situation
User sering membutuhkan konversi file untuk kebutuhan sehari-hari:
- Mahasiswa perlu mengkonversi PDF ke Word untuk editing tugas
- Pekerja kantor perlu mengkonversi gambar untuk presentasi
- Freelancer perlu kompres gambar untuk upload ke platform
- Content creator perlu mengubah format gambar untuk berbagai platform

Saat ini, user harus:
1. Menggunakan tool berbayar (Adobe Acrobat, Canva Pro)
2. Menggunakan situs converter gratis yang penuh iklan
3. Install software tambahan (Photoshop, GIMP)
4. Mengorbankan privasi (file diupload ke server pihak ketiga)

## The Problem
**Tidak ada file converter yang benar-benar gratis, aman, dan mudah digunakan untuk user Indonesia.**

Masalah spesifik:
1. **Batasan:** Tool gratis biasanya memiliki limit harian (5-10 konversi/hari)
2. **Iklan:** Situs converter gratis dipenuhi iklan mengganggu yang menutupi konten
3. **Privasi:** File sensitif harus diupload ke server yang tidak diketahui keamanannya
4. **Kualitas:** Hasil konversi seringkali buruk (format rusak, gambar pecah)
5. **Aksesibilitas:** Tidak semua user memiliki akses ke software berbayar

## Impact
- **Productivity Loss:** User menghabiskan 15-30 menit mencari tool yang works
- **Privacy Risk:** File sensitif (KTP, kontrak, data keuangan) terpapar ke pihak ketiga
- **Financial Burden:** Software berbayar tidak terjangkau untuk mayoritas user Indonesia
- **Frustration:** Iklan dan limit harian menyebabkan pengalaman yang buruk

## Proposed Solution (High Level)
**Gantiin** — Web application gratis yang memproses file konversi langsung di browser user (client-side), sehingga:
1. **Gratis tanpa batas** — Tidak ada limit harian atau registrasi wajib
2. **Privasi terjamin** — File tidak pernah meninggalkan browser user
3. **Tanpa iklan mengganggu** — UI bersih dan fokus pada konversi
4. **Mudah digunakan** — Cukup drag & drop, pilih format, download
5. **Mobile-friendly** — Bisa digunakan dari smartphone kapan saja
